# rose-server
> Rose server with NodeJS.

##### NOTE : This project need node version >= 6

This project uses

* ES6 ( Using babel-register )
* NodeJS ( version >= 6 )
* Express
* Mongoose
* Body-parser
* Socket.io

### Get Started
Fire up your terminal
```sh
git clone https://github.com/raylansoares/rose-server.git
cd rose-server
npm install
```

Start server by typing
```sh
npm start
```
